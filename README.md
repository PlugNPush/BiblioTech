# BiblioTech
Master Project M1

# Executer le front :
- se placer dans le dossier my-app
- exécuter les lignes :
```zsh
npm i
npm run start
```

# Executer le back :
- set up la bdd (voir fichier README.md dans le dossier server)
- se placer dans le dossier server
- exécuter dans un terminal la ligne :
```zsh
npm i
npm run start
```
